
## 安装

### 环境要求

* PHP >= 7.0
* swoole >= 1.8.9

### 安装


1、composer

```
composer require kcloze/swoole-bot
```

2、git

```
git clone https://github.com/kcloze/swoole-bot.git
cd swoole-bot
composer install
```

然后执行``` php src/start.php ``` 



### 说明
* 该项目主要目的想利用swoole，增强原生PHP cli的能力
* 目前没考虑修改原作者的底层封装，后期可考虑用swoole task增强效率
